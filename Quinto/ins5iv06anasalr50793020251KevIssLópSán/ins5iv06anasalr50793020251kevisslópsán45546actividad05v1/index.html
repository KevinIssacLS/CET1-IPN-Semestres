Actividad 05

CLASIFICACIÓN DE SOFTWARE DE ACUERDO A SU USO

Instalación de Hardware

Es el proceso de conectar y configurar componentes físicos (periféricos o internos) a una computadora o red, asegurando que funcionen correctamente.

Pasos principales

  • Conectar correctamente el dispositivo a la fuente de energía y al equipo (tarjetas gráficas, discos duros, impresoras, etc.).
  • Verificar la compatibilidad del hardware con el sistema operativo.
  • Instalar los controladores (drivers) necesarios para que el hardware funcione.
  • Configurar los ajustes en el sistema operativo o la BIOS para reconocer el nuevo hardware.
  • Realizar pruebas de funcionamiento.


Instalación de Software

Es el proceso de copiar los archivos de un programa o aplicación en una computadora y configurarlo para que funcione correctamente.

Pasos principales

  • Descargar o adquirir el software.
  • Verificar los requisitos del sistema (memoria, procesador, sistema operativo, espacio en disco).
  • Ejecutar el instalador y seguir las instrucciones.
  • Configurar los parámetros de instalación (idioma, ubicación, etc.).
  • Registrar o activar el software si es necesario.


Licencias y sus clasificaciones

  • Libre: El software es de código abierto, y cualquier persona puede usar, modificar y distribuir el programa sin costo. Ejemplo: Linux.
  • Comercial: Es un software que tiene un costo y una licencia para usarlo, y generalmente su código no es accesible. Ejemplo: Microsoft Office.
  • No comercial: Software utilizado con fines educativos o personales, sin fines de lucro. A veces tiene ciertas restricciones en comparación con la versión comercial.
  • Nube: Software que no se instala localmente, sino que se accede a través de Internet. Ejemplo: Google Drive, Office 365.


Tipos de Software

  • Shareware: Software distribuido de manera gratuita por un tiempo limitado o con características restringidas. Ejemplo: WinRAR.
  • Freeware: Software que es completamente gratuito, pero su código fuente no está disponible para ser modificado. Ejemplo: Adobe Acrobat Reader.
  • Propietario: El software cuyo código fuente no está disponible, y los usuarios solo tienen permiso de usarlo bajo términos de licencia. Ejemplo: Windows.
  • Demostración: Software de prueba con funcionalidades limitadas o un periodo de uso limitado para que los usuarios lo evalúen. Ejemplo: Photoshop (versión trial), vMix(versión de prueba).
  • Dominio Público: Software sin derechos de autor, disponible para ser usado y modificado por cualquier persona. Ejemplo: el software CP/M.
  • Especificación: Software que está desarrollado para un uso muy específico, como aplicaciones empresariales personalizadas. Ejemplo: Titler Live,
  • Sistema: Incluye el sistema operativo (como Windows, Linux, macOS) y software de gestión de datos como bases de datos (MySQL, Oracle).
  • Malicioso: Software diseñado con la intención de dañar o comprometer un sistema. Ejemplo: Virus, troyanos.
  • Control de dispositivo: Software que permite que el sistema operativo controle dispositivos de hardware. Ejemplo: Drivers de impresoras o tarjetas de video.
  • Código abierto: El software cuyo código fuente está disponible para que los usuarios lo modifiquen y distribuyan. Ejemplo: OpenOffice, OBS.


Instalación de Red

La instalación de red es el proceso de configurar y conectar equipos para que puedan comunicarse entre sí e intercambiar datos.

Pasos principales

  • Diseñar la red y determinar el tipo de red (LAN, WAN, Wi-Fi, etc.).
  • Configurar los dispositivos de red (routers, switches, puntos de acceso).
  • Asignar direcciones IP a los dispositivos.
  • Configurar la seguridad de la red (contraseñas, cifrado, firewall).
  • Probar la conexión entre los dispositivos para asegurar la comunicación.
