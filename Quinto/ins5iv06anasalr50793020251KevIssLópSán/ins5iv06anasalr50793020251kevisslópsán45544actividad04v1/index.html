Actividad 04

SOFTWARE

Definición

El software es el conjunto de programas, aplicaciones y datos que permiten a una computadora realizar tareas específicas. Es lo que controla y guía el funcionamiento del hardware, haciendo posible la interacción entre el usuario y la máquina.


Clasificaciones de Software

Sistemas Operativos

Los sistemas operativos son el software fundamental que gestiona los recursos de hardware y permite la comunicación entre los programas y los componentes físicos del equipo. Ejemplos incluyen Windows, macOS, Linux y Android.

Desarrollo

El software de desarrollo incluye herramientas y programas utilizados para crear otros programas, aplicaciones o sistemas. Entre estos están los entornos de desarrollo integrado (IDEs), compiladores y editores de código, como Visual Studio, Eclipse o Git.


Diseño

El software de diseño se utiliza para crear gráficos, animaciones, modelos 3D, entre otros trabajos visuales. Programas como Photoshop, Illustrator o AutoCAD son ejemplos de software de diseño.


Aplicaciones

Las aplicaciones son programas diseñados para realizar tareas específicas para el usuario final, como procesamiento de texto, hojas de cálculo, navegación por internet o mensajería. Ejemplos incluyen Microsoft Word, Excel o Google Chrome.

Seguridad

El software de seguridad está diseñado para proteger los sistemas informáticos y los datos de los usuarios contra amenazas como virus, malware, accesos no autorizados y ataques cibernéticos. Este tipo de software incluye programas como antivirus, firewalls, antispyware, y herramientas de cifrado, entre otros.

  • Antivirus: Detecta y elimina virus y malware.
  • Firewall: Controla el tráfico de red entrante y saliente para evitar accesos no autorizados.
  • Antispyware: Protege contra software espía que intenta recopilar información del usuario sin su consentimiento.
  • Cifrado: Protege datos sensibles codificándolos para evitar accesos no deseados.


Ejemplos comunes de software de seguridad incluyen Norton, McAfee, Bitdefender y Kaspersky.


Pasos de Instalación de un Sistema Operativo:

Preparación

  • Verificar los requisitos del sistema (procesador, memoria, espacio en disco, etc.).
  • Respaldar la información importante.
  • Obtener el medio de instalación (USB, CD, o descarga del sistema operativo).

Iniciar la instalación

  • Insertar el medio de instalación (USB, CD).
  • Reiniciar la computadora y acceder al menú de arranque (boot) para seleccionar el medio de instalación.

Configurar instalación

  • Elegir el idioma, zona horaria y configuración del teclado.
  • Seleccionar si deseas instalar el sistema operativo en un disco existente o particionar el disco duro.

Instalación del sistema operativo

  • Formatear la unidad de almacenamiento si es necesario.
  • Seguir las indicaciones para instalar el sistema operativo en el disco elegido.
  • Esperar a que el proceso de instalación copie archivos y configure el sistema.

Configuración inicial

  • Una vez instalada, el sistema pedirá configuraciones como la creación de un usuario, contraseña, ajustes de red, etc.
  • Instalar controladores o actualizaciones si es necesario.

Finalización

  • Reiniciar el equipo y verificar que el sistema operativo funcione correctamente.
  • Instalar el software adicional y restaurar archivos si se hizo un respaldo.
