Ejercicios 4 y 5

EJERCICIO 04

¿Qué debemos hacer para que no perdamos nuestros datos?

  • Para evitar la pérdida de datos, es importante hacer copias de seguridad regulares, proteger nuestras cuentas con contraseñas seguras y únicas, y utilizar medidas de autenticación de dos factores (2FA) cuando esté disponible. Además, debemos evitar el acceso a enlaces sospechosos y verificar la seguridad de las aplicaciones que instalamos en nuestros dispositivos


¿Qué herramientas de seguridad podemos instalar en nuestro dispositivo?

  • Antivirus: Protege contra programas maliciosos como virus, troyanos y spyware, que pueden dañar el dispositivo o robar información personal.
  • VPN (Red Privada Virtual): Ayuda a proteger nuestra información en redes públicas o desconocidas al cifrar la conexión, lo cual dificulta que otros intercepten nuestros datos​


¿Cuáles son los riesgos en una red wifi pública? ¿Qué medidas de seguridad debemos aplicar?

  • En redes wifi públicas, los riesgos incluyen la posibilidad de que otros usuarios intercepten nuestra información o accedan a nuestros dispositivos. Para reducir estos riesgos, se recomienda no ingresar información sensible, usar una VPN para cifrar la conexión y desactivar el uso compartido de archivos y dispositivos


¿Qué puedes hacer para que nadie averigüe tus contraseñas?

  • Para mantener seguras nuestras contraseñas, se recomienda crear combinaciones únicas de al menos 12 caracteres que incluyan letras mayúsculas, minúsculas, números y símbolos. También es útil usar un gestor de contraseñas para almacenar y generar contraseñas seguras, y habilitar la autenticación de dos factores


Creación de una contraseña segura y explicación del proceso:

  • Un ejemplo de contraseña segura podría ser algo como Mt$4l#pX9z!3. Para crearla, usé un método de sustitución y combinación de caracteres:
  1. Elegí una frase corta fácil de recordar y reemplacé algunas letras con números y símbolos.
  2. Incluí una mezcla de mayúsculas, minúsculas y caracteres especiales para que sea más difícil de adivinar.
  3. Me aseguré de que la longitud fuera suficiente para resistir ataques de fuerza bruta.


EJERCICIO 05

¿En qué consiste el phishing?

  • El phishing es una técnica de fraude en la que los atacantes se hacen pasar por una entidad confiable, como un banco o una red social, para engañarnos y obtener información sensible, como contraseñas o datos bancarios. Este engaño suele ocurrir a través de correos electrónicos o mensajes que contienen enlaces a sitios falsos que imitan a los reales​.


¿Cómo podemos añadir una capa de seguridad extra a nuestra cuenta?

  • Podemos añadir una capa adicional de seguridad activando la autenticación de dos factores (2FA). Esto requiere que, además de nuestra contraseña, ingresemos un código de verificación enviado a nuestro teléfono o generado en una aplicación de autenticación, lo que dificulta el acceso a los intrusos incluso si conocen nuestra contraseña​.


¿Cómo podemos obtener los códigos de seguridad?

  • Los códigos de seguridad suelen ser generados por aplicaciones de autenticación (como Google Authenticator o Authy) o enviados por SMS al número telefónico asociado a nuestra cuenta. Este método de generación es importante para autenticación de dos factores​.


¿Por qué debemos realizar copias de seguridad?

  • Las copias de seguridad nos permiten proteger nuestra información ante posibles fallos en el dispositivo, pérdidas o ataques de malware. Tener copias de seguridad actualizadas facilita la recuperación de datos importantes en caso de un problema​.


Consejos y recomendaciones para no perder la información

  • (Aquí se sugiere transcribir en el cuaderno algunos puntos clave que podrían incluir: hacer copias de seguridad regularmente, usar contraseñas seguras, mantener el sistema actualizado, no compartir datos sensibles y usar autenticación en dos pasos).


¿Cuáles son los riesgos de los trámites online?

  • Al hacer trámites en línea, pueden existir riesgos como la exposición a sitios web falsos, robo de información personal, y ataques de malware. Estos riesgos aumentan si no se confirma la autenticidad de los sitios o si se usan redes públicas para trámites confidenciales​.


¿Cómo podemos proteger adecuadamente nuestro dispositivo?

  • Podemos proteger nuestro dispositivo instalando un buen antivirus, manteniendo el sistema y aplicaciones actualizadas, y configurando contraseñas o bloqueos de pantalla. También es recomendable desactivar la conexión automática a redes públicas y usar aplicaciones de seguridad adicional si es posible​.


¿Cómo podemos asegurarnos de estar en la web correcta? ¿Qué es la URL?

  • La URL es la dirección web que aparece en la barra del navegador. Para asegurarnos de estar en el sitio correcto, debemos revisar la URL y confirmar que comience con “https” (lo que indica que es segura) y que el dominio sea el oficial del servicio que buscamos. Evitar enlaces de correos sospechosos también es esencial​.


¿Qué debemos tener en cuenta si compramos online?

  • Es importante verificar que el sitio web sea seguro y confiable, leer reseñas de otros usuarios, evitar realizar pagos desde redes públicas y usar métodos de pago seguros. Asegurarnos de que la URL comience con “https” también ayuda a confirmar que los datos estarán protegidos durante la compra
